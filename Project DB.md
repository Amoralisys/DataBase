Требование к проекту “База данных”
Реализовать на языке с++ программу для хранения таблиц.
В реализации требуется:

1) ввод/вывод таблицы из консоли
2) ввод/вывод таблицы из файла
3) открытие файла с существующей таблицей или создание нового файла для записи в него
4) запросы на поиск в таблице по полям и вывод результатов на экран или в файл
5)сортировка таблицы по заданному полю по возрастанию/убыванию
6)добавление данных в таблицу
7) удаление данных из таблицы
8) обновление таблицы
